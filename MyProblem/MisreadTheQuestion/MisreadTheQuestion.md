# MisreadTheQuestion

## 题目背景

## 题目描述

给定一个 $2$ 行 $n$ 列的网格 $a$，每个格子内有一个数字，记第 $i$ 行第 $j$ 列中的数字为 $a_{i,j}$。

定义 $\operatorname{f}(l, r)\,(1 \le l \le r \le 2n)$ 表示一个二进制的 $2$ 行 $n$ 列网格 $b$，当且仅当 $l \le a_{i,j} \le r$ 时，$b_{i,j} = 1$。

请计算满足条件的整数对 $(l, r)$ 的数量，其中 $1 \le l \le r \le 2n$，并且在 $f(l, r)$ 中存在一条从单元格 $(1, 1)$ 到单元格 $(2, n)$ 的**四连通**路径，满足路径上每个单元格的值均为 $1$。

> 1. 一个网格是二进制的，当且仅当它的每个单元格的值都是 $0$ 或 $1$。
>
> 2. “单元格 $(x,y)$”代表位于第 $x$ 行第 $y$ 列的单元格。
>
> 3. 对于两个单元格 $(x,y),(i,j)$，它们四连通当且仅当 $|x-i|+|y-j|=1$。

## 输入格式

<!-- 每个测试包含多个测试用例。第一行包含测试用例的数量 $t$。测试用例的描述如下。 -->

第一行输入一个整数 $n$ 表示网格的列数。

第二行输入 $n$ 个整数，第 $i$ 个整数表示 $a_{1,i}$。

第三行输入 $n$ 个整数，第 $i$ 个整数表示 $a_{2,i}$。

## 输出格式
输出一个整数表示满足条件的整数对 $(l, r)$ 的数量。
